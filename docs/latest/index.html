排错速查

Rust bot 没产 startup_*.json？

tie-out 会直接报 No startup snapshots — nothing to compare。 去 poly_temp_bot/state/bitcoin_5min/ 看是否有 startup_*.json；没有的话 Rust bot 当天没启动。

frozen-tag 找不到？

tie-out 会按 startup 当天的本地日期去 signal_dashboard/<signal>/output/<date>/user_tag.parquet 找 daily snapshot。如果主 dashboard 当天没跑成功，就需要去补跑 python signal_dashboard/<signal>/run_signal.py。

activity API 一直空？

确认 funder address (config.yaml > funder_address) 和 online_pnl/calc_pnl.py 里的 ADDRESS 一致。 Rust 钱包是 brand new，前几天可能本来就没几笔交易。

SR 不一致（散点偏离 y=x）？

如果 Pearson r < 0.9 或 offset 明显，说明 Rust bot SR 计算或 smart-maker tag 加载有 bug。先检查 user_tag_current.json 的 max_date 和 n_smart_makers 是否合理。
